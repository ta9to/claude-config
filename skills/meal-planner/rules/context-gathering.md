## 質問プロトコル

### 最初の質問（必須・セットで聞く）

```
直近2〜3日で食べたものを教えてもらえますか？
（「昨日は外食でラーメン、一昨日は家で焼き魚定食」みたいにざっくりでOKです）

あと、今日はどんなものが食べたい気分ですか？
```

---

### 2回目の質問（補足情報）

直近の食事と気分がわかったら、次にまとめて聞く。

```
ありがとうございます。あといくつか確認させてください。

- 何人分ですか？（例：大人2人、子ども1人）
- 予算感はどのくらいですか？（例：3日で3,000円くらい）
- 調理にかけられる時間はどのくらいですか？（例：平日は30分、休日は余裕あり）
- 苦手な食材やアレルギーはありますか？
```

すべてを答えてもらえなくてもよい。「わからない」「なんでもいい」はデフォルト値として扱う。

---

## デフォルト値（未回答時）

| 項目 | デフォルト値 |
|------|-------------|
| 人数 | 2人（大人2人） |
| 予算 | 2〜3日で3,000〜4,000円 |
| 調理時間 | 30〜40分 |
| 食の制限 | なし |
| 朝昼の対応 | 夕食のみ計画（朝昼は簡単に済ませる前提） |

---

## 気分の解釈ガイド

ユーザーの「気分」の発言から、好みのジャンルや料理スタイルを推測する。

| 発言 | 推測 |
|------|------|
| 「さっぱりしたい」「重いものは嫌」 | 和食・蒸し料理・サラダ系 |
| 「がっつり食べたい」「肉！」 | 炒め物・焼き肉・丼系 |
| 「温かいものが食べたい」「ほっこりしたい」 | 煮物・鍋・シチュー系 |
| 「簡単なものでいい」「疲れてる」 | 調理工程が少ないもの・1品完結系 |
| 「珍しいものが食べたい」「気分転換したい」 | 普段と異なるジャンルや食材 |
| 「ヘルシーにしたい」「食べ過ぎた」 | 野菜多め・低カロリー・和定食系 |

---

## 注意事項

- 食事履歴は記憶しない（セッション内の発言のみを使う）
- 子どもがいる場合は辛さや味の濃さを控えめに調整する
- アレルギーは絶対に無視しない。代替食材が存在する場合は提示する
